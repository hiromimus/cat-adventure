🐱 ネコの冒険 ～フィッシュラン～
3Dで描かれた豊かな自然の中を駆け抜けるアクションゲームです。 お魚を集め、敵を避けながらハイスコアを目指しましょう！

🎮 操作方法
このゲームはキーボード、およびゲームパッド（コントローラー）での操作に対応しています。

キーボード操作
移動: W A S D または 矢印キー

ジャンプ: SPACE

ダッシュ: SHIFT を押しながら移動

コントローラー操作
移動: 左スティック

ジャンプ: Aボタン（または下側のボタン）

ダッシュ: Bボタン / R1 / R2ボタン

🐟 得点システム
お魚の獲得: お魚を拾うとスコアが加算されます。

👾 敵の撃破: 敵を上から踏みつけることで倒すことができ、ボーナススコアが入ります。

ペナルティ: 敵に横から当たるとスコアが減り、ライフが減少します。

ステージの変化: 500点ごとに世界の色が変わり、敵の増援が現れます！
